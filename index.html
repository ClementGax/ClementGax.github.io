A propos

Bonjour ! Je suis Clément, un jeune développeur web basé dans le Grand-Est.

Après une carrière dans le domaine assurantiel, j'ai effectué une reconversion professionnelle dans le développement web pour suivre mes rêves.

Pour réussir ce pari osé, j’ai suivi la formation proposée par l’école O’Clock (labelisée grande école du numérique). Dorénavant je recherche une opportunité afin de finaliser ce beau projet.

En 2021 c'est décidé, je deviens Développeur web !
